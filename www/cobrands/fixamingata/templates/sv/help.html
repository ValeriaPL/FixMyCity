Om <% print( CONFIG.APP_NAME ) %>

Vad är <% print( CONFIG.APP_NAME ) %>?
<% print( CONFIG.APP_NAME ) %> är en app för att underlätta rapportering av problem i gatumiljön till kommuner.
Hur används appen?
Appen kommer att försöka hitta din position automatiskt, alternativet är att söka på postnummer eller plats. För att rapportera ett problem, navigera kartan så att platsen med problemet lokaliseras och klicka på 'Ny rapportering här'.
Vad händer med informationen som jag anger?
All information som du anger kommer att skickas till kommunen. Din rapports titel, kategori och detaljer kommer att visas offentligt på <% print( CONFIG.APP_NAME ) %>. Som standard visas också ditt namn på <% print( CONFIG.APP_NAME ) %>, men du kan göra ett aktivt val för att dölja uppgiften. Din e-postadress eller ditt telefonnummer kommer aldrig visas offentligt på <% print( CONFIG.APP_NAME ) %>.
Hur kan jag välja att vara anonym?
Avmarkera 'Visa mitt namn offentligt' när du fyller i dina uppgifter så kommer ditt namn inte att visas på <% print( CONFIG.APP_NAME ) %>. Ditt namn, din e-postadress och ditt telefonnummer (om du anger det) kommer alltid att skickas till kommunen.
Vad gör jag om jag om personlig information angivits i en rapport av misstag?
Kontakta <% print( CONFIG.APP_NAME ) %>-supporten på info@fixamingata.se så hjälper vi dig.
Hur kan jag återställa mitt lösenord?
Om du har glömt bort ditt lösenord kan det återställas genom att välja 'Sätt ett lösenord'-alternativet vid nästa tillfälle en rapport skapas. Notera att du alltid måste klicka på länken i bekräftelsemejlet för att aktivera eller ändra ditt lösenord.
Vilken sorts problem skall jag rapportera med <% print( CONFIG.APP_NAME ) %>?
<% print( CONFIG.APP_NAME ) %> är huvudsakligen till för att rapportera saker som är trasiga, smutsiga, skadade, tippade, eller behöver lagas, rensas eller städas upp.
Vad är <% print( CONFIG.APP_NAME ) %> inte till för?
<% print( CONFIG.APP_NAME ) %> är inte till för att kontakta din kommun i andra ärenden än ovan. Vänligen använd bara <% print( CONFIG.APP_NAME ) %> för ärenden som kan kopplas till angivna kategorier.
Är <% print( CONFIG.APP_NAME ) %> gratis?
Webbplats och app är gratis att använda.
Vilka ligger bakom <% print( CONFIG.APP_NAME ) %>?
<% print( CONFIG.APP_NAME ) %> är en e-tjänst som drivs av Föreningen Sambruk och är framtagen i samarbete med KIVOS och FFKP. <% print( CONFIG.APP_NAME ) %> är baserad på FixMyStreet, ett projekt skapat av brittiska mySociety.
Tack till...
OpenStreetMap och MapQuest (för kartorna), fri programvarurörelsen (detta projekt använder Phonegap, Backbone, och jQuery Mobile) och FixMyStreet. Säg till om vi har missat någon.
